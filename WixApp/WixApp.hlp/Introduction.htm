Dialog App

Introduction

WixApp is an example of a Windows Dialog application. It includes a trimmed down MFC starting application InitInstance. The main dialog has a menu, a toolbar with various features and a status bar on the bottom of the client part of the window. The main dialog has a fixed size and may be populated with all the usual controls that are available in Visual Studio 17's Resource editor.

There is a lot of functionality buried in the MainFrame of the normal MFC application. For example the tool bar depends heavily of the MainFrame. This example has a tool bar but and status bar but the functionality is a subset of that available in a normal MFC application. The operations available are discussed on the operations page.
