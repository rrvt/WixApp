SFTP Operations

New Web Site

Bring up a dialog box to create a representation of a web site suitable for managing the transfer of files between the PC and the web host.

Pick the Web Site to Manage

Lists all the web sites that are managed on this PC and allows the user to pick one.

Edit the Current Web Site Details

Brings up the dialog box with the currently selected web site so that the details may be changed.

Compare Web Host and Local PC Files

Determine the list of directories and files on the web host and compare them to the local PC files. Using the presence and absense of directories and files in those list determine the files to upload to the host and download to the PC. Mark those files appropriately. The actual transfers are done by the Update command. This could take some time to download the Web Host details so a progress bar is shown during the download.

Compare Local PC Files Only

Each time an Upload occurs the size and date of the local PC files is saved. When a change to a file is made the date and/or the size of a local PC file changes. This comparison is used to determine the files to be marked for uploading to or deleting from the web host. Also the present and absence of directories in the last upload list and the current entity list will determine which directories to create or remove from the web host.

Update Web Host and/or Local Directory and or Files

A dialog box is presented with a list of operations and entity descriptions. A check box accompanies each line. The user must indicate which operations are to be performed by selecting one or more check boxes.

Display the Web Host Directories and Files

Produces an indented list of directories and files to represent the files as they appear on the Web Host. This might take some time to download the information so a progress bar is shown during the download.

Save File

Displays a Save File Dialog and saves the content of the window to the file to the file named.

Print

Will do all the normal things to print the content of the window.

Options

The options dialog allows specifying the Display and Printer Font Size Scale Factor, the printer orientation and the printer margins.

About

Displays the About Box. The about box uses text from the resource file (TestApp.rc) for the output information. There is a standard about box dialog in the Library.

Exit

Exit the application with no reservations.

Menu: File

Delete Site Details

When a Web Site becomes redundant one may wish to remove the details of that web site from this application. This is done by Opening the web site normally either by using the Pick Web Site or because it is the default web site. Then choose this command to delete all references to it from this application.

A dialog box will appear to be sure you really mean to delete it, answer yes and it will be gone.

Menu: Test

This menu provides a list of low level SFTP commands that are used to test the interface between the PC and a specific Web Host. They are left in to allow exploration as things change.
